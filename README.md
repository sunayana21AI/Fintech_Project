# Bluestock Mutual Fund Analytics Project

## Overview

This project analyzes Indian Mutual Fund industry data using Python, SQL, SQLite, and Power BI.

## Features

* Data Ingestion & Cleaning
* SQLite Database Design
* Exploratory Data Analysis (EDA)
* Fund Performance Analytics
* Risk Metrics & Advanced Analytics
* Interactive Power BI Dashboard

## Tech Stack

* Python
* Pandas
* NumPy
* SQL
* SQLite
* Power BI
* Plotly
* Matplotlib
* Seaborn
